Help

คำแนะนำการจัดการไอคอน

  • ขนาดที่แนะนำ: ควรใช้รูปจัตุรัส (1:1) เช่น 128x128 px เพื่อความสวยงาม
  • นามสกุลไฟล์: รองรับ PNG (แนะนำแบบโปร่งใส), JPG และ WebP
  • การแสดงผล: หลังจากกดบันทึก ระบบจะอัปโหลดไฟล์เข้า Supabase Storage และนำ URL มาใช้งานแทนไฟล์เดิมในเครื่องทันที
  • Dashboard Icons: หากไม่ได้อัปโหลดรูป ระบบจะใช้ไอคอนมาตรฐาน (SVG) ที่เตรียมไว้ให้
